🚀 NovaStyle – Front-End Fashion E-commerce Demo

Overview:
A fully responsive front-end only fashion e-commerce website demo built with HTML, CSS (Bootstrap), and JavaScript. This project showcases UI design, interactive animations, and a modern layout for fashion collections.

Live Demo: https://nova-style-two.vercel.app/

 Features:
- Hero video with scroll-scale text animation  
- Best-seller and trending product sections  
- Seasonal collections and campaign highlights  
- TikTok-style inspiration carousel  
- Fashion blog/journal section  
- Newsletter subscription form  
- Interactive footer with accordion menus  
- Smooth animations using AOS.js  
- Fully responsive across devices

Tech Stack:
- HTML5  
- CSS3 / Bootstrap 5  
- JavaScript (Vanilla)  
- AOS.js for animations

Note:
This is a front-end only demo; no backend, database, or payment integration.
